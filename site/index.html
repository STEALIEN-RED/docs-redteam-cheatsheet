    Red Team Cheat Sheet

    레드팀 및 모의침투 업무에서 사용하는 명령어, 기법, 도구를 정리한 문서.

    실제 공격 라이프사이클 흐름에 맞춰 분류했고, 실무에서 바로 참고할 수 있도록 명령어 중심으로 구성했다.

    공격 라이프사이클

    graph LR
    A[외부 정찰] --> B[초기 침투]
    B --> C[내부 정찰/열거]
    C --> D[자격 증명 탈취]
    D --> E[권한 상승]
    E --> F[횡적 이동]
    F --> G[지속성 유지]
    F --> C
    E --> C
    단계 설명
    외부 정찰 OSINT, 포트 스캔, 서비스 식별, 디렉토리 열거
    초기 침투 피싱, Password Spray, 외부 서비스 취약점
    내부 정찰 및 열거 SMB, LDAP, RPC, AD 환경 열거
    자격 증명 탈취 Kerberoasting, AS-REP Roasting, DCSync, LLMNR Poisoning
    권한 상승 Windows/Linux 로컬 권한 상승, AD DACL Abuse
    횡적 이동 PtH, PtT, WinRM, PSExec, DCOM
    지속성 유지 레지스트리, Scheduled Task, Golden Ticket

    프로토콜별 펜테스트

    포트 스캔 후 발견된 서비스에 대한 프로토콜별 공격 가이드.

    프로토콜 포트 설명
    SMB 445 파일/프린터 공유, AD 열거, PtH
    LDAP 389/636 디렉토리 서비스, AD 쿼리
    HTTP 80/443 웹 서비스, 디렉토리/파라미터 열거
    WinRM 5985/5986 PowerShell Remoting
    SSH 22 Linux/Unix 원격 접근, 터널링
    FTP 21 파일 전송, Anonymous 접근
    RDP 3389 Windows 원격 데스크톱
    DNS 53 Zone Transfer, 서브도메인 열거
    Kerberos 88 사용자 열거, AS-REP/Kerberoasting
    RPC 135/111 MSRPC 열거, rpcclient
    MSSQL 1433 xp_cmdshell, NTLM 캡처
    MySQL 3306 UDF, 파일 읽기/쓰기
    SNMP 161 Community String, 정보 수집
    NFS 2049 마운트, root squashing 우회
    SMTP 25 사용자 열거, 메일 스푸핑

    기술별 분류

    분류 설명
    AD 환경 공격 인증 프로토콜별 공격, Delegation, NTLM Relay, Trust
    ADCS ESC1~ESC16 인증서 템플릿 공격
    Web 공격 SQLi, XSS, SSRF, SSTI, JWT, Deserialization
    Cloud 공격 AWS, Azure 클라우드 환경 공격
    방어 우회 AV/EDR 우회, AMSI bypass, AppLocker bypass

    오퍼레이션 인프라

    항목 설명
    리버스 쉘 다양한 언어/환경별 쉘 페이로드
    파일 전송 HTTP, SMB, SCP, Base64 인코딩 전송
    피봇 / 터널링 Ligolo-ng, Chisel, SSH, sshuttle
    C2 프레임워크 Sliver, Havoc, Cobalt Strike
    도구 레퍼런스 주요 도구별 명령어 정리
